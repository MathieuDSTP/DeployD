DeployD.

DeployD is an open source automated deployment framework for Windows. It's designed to make Windows deployments as easy as package management and publishing.

Built on NancyFx and NuGet

We're building on top of some cool open source tech to get this finished quickly. We love capistrano, appharbor and heroku and want the same kind of deployment story in our own private hosted environments.

Simple Set Up

We don't want to have huge server farm managing applications. We believe each node should stand alone, and support it's own tear up and tear down. Each node has it's own API so you can manage things remotely.

Built in hooks for common tasks

Web.config transformations? Got it. IIS management? Got it. Powershell hooks? Got it. We want to make it as easy as possible to bend to your infrastructure without friction.

Simple configuration

Configuration is just another published package! All the agents will automatically synchronise with any configuration package changes and update themselves accordingly.
